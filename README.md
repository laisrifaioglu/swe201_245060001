## Project Description
This project is a simple C# Console Application required for the midterm assignment.  
It contains one custom class called `Product` and a `Program` class with the `Main` method.

### Product.cs includes:
- Two member variables (`name`, `price`)
- Two properties (`Name`, `Price`)
- An empty constructor
- A constructor with parameters
- A member method `ApplyDiscount(double percentage)`
- An overridden `ToString()` method to print product details

### Program.cs includes:
- Creating objects from the `Product` class
- Printing object information using `ToString()`
- Calling the `ApplyDiscount` method
- Showing discounted prices

## How to Run
1. Open a C# IDE (Visual Studio, Rider, or VS Code with C# extension).
2. Create a new **Console App** project.
3. Add the two files: `Product.cs` and `Program.cs`.
4. Build and run the program (Ctrl + F5).
